# Next.js Blog
Next.js build simple project blog with json-server

## Installation

<pre>
    1. cd server/
    1. npm install 
</pre>

## Start server

<pre>
    1. cd server/
    1. npm run dev -- -p 3001
    2. json-server resources/data/db.json --port 3002
</pre>

## Open App

<pre>
    Open app <a href="http://127.0.0.1:3001">http://127.0.0.1:3001</a>
</pre>
